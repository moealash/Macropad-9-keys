# 9 Key Macropad

## Project Overview
This is a 9 key macropad that I designed using KiCad.  
A macropad is like a small keyboard that can run shortcuts or macros when you press a button.

My macropad has a 3×3 layout (9 buttons total). When you press a key it sends a command to the computer.

## Features
9 mechanical keys
Custom PCB made in KiCad
Small 3×3 layout
Can run macros or shortcuts
Connects to a computer with USB

## Hardware
Parts used in this project:

XIAO RP2040 microcontroller
9 Cherry MX switches
9 keycaps
Diodes
Custom PCB

## How It Works
Each key is connected to a pin on the microcontroller.  
When a key is pressed, the microcontroller detects it and sends a keyboard signal to the computer.
